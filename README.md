# Psi Engine

### Components:
###### Sol Library
A collection of (mostly abstract) classes and methods that can be reused in various games.

###### Generic Shooter Game
A generic shooter game. [WIP]

### Dependencies:
CMake
Boost
GLFW

### Building:
###### UNIX:
    mkdir build
    cd build
    cmake ..
    make

Binary GSG will be stored in build/generic_shooter_game.
